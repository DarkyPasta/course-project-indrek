1. Project Idea and Purpose
The purpose of this project was to create a multi‑page responsive portfolio website using HTML and CSS. The website showcases my skills learned during the course, including layout techniques, responsive design, and clean coding practices. The site includes four pages: Home, About, Projects, and Contact.

The goal was to build a visually appealing and mobile‑friendly website that demonstrates my ability to structure content, style pages, and create a consistent user experience.

2. Tools and Technologies Used
HTML5 for page structure

CSS3 for styling and layout

Flexbox & CSS Grid for responsive layouts

Media Queries for mobile optimization

VS Code as the development environment

Live Server for real‑time preview

W3C HTML Validator to ensure valid markup

W3C CSS Validator to ensure valid styling

3. Challenges and How I Solved Them
Responsive Design
Ensuring the layout worked on both desktop and mobile required adjustments to grid and flexbox settings. I solved this by using auto-fit, minmax(), and media queries.

Navigation Consistency
I needed to ensure the navigation bar worked across all pages. I solved this by copying the same header structure into each HTML file and updating the active link.

Background Styling
When adding a gradient background, text visibility became an issue. I fixed this by adding semi‑transparent white backgrounds to content sections and adjusting text colors.

File Structure
To keep the project organized, I created a dedicated course-project folder with separate HTML files, a shared CSS file, and an images folder.

4. Additional Features and Improvements

Styled hero and content sections with semi‑transparent backgrounds

Ensured all pages are fully responsive

Improved readability with consistent spacing and typography

Added hover effects and visual polish

Ensured all code is fully valid according to W3C validators

5. Conclusion
The project demonstrates my ability to build a complete, responsive, multi‑page website using HTML and CSS. I applied all key concepts from the course and improved the design with additional styling techniques. The final result is a clean, functional, and visually appealing portfolio website.
